# factline.one

The decentralized, static public dashboard for the Factline platform.

This repository contains the public-facing React application that transparently displays investigations, claims, and verifiable evidence generated by the core **Factline AI** intelligence engine.

## Overview

Factline is a human-guided intelligence investigation platform. While the core engine performs the automated planning, evidence collection, and report generation, this dashboard serves as the public interface where those reports are hosted.

- **Main Repository:** [factline_ai](https://github.com/pratyakshkwatra/factline_ai)

## Architecture

This frontend is a static React application built with Vite and TailwindCSS. It fetches detailed JSON payloads generated by the intelligence engine, ensuring lighting-fast load times and complete transparency.

## Getting Started

### Prerequisites
- Node.js (v18+)

### Installation

```bash
git clone https://github.com/pratyakshkwatra/factline.one.git
cd factline.one
npm install
```

### Running Locally

```bash
npm run dev
```

The dashboard will be available at `http://localhost:5174` (or similar port).
